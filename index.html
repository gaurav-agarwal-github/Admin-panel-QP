GitHub Settings

Your token is stored only in this browser (localStorage). It is never sent anywhere except directly to GitHub API over HTTPS.

To create a token: GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic) → Generate new token → check repo scope only.

Push to Live Website

This will update script.js in your GitHub repo (gaurav-agarwal-github/Quality-Pharmaceuticals, branch main). Vercel will automatically redeploy your site within ~30 seconds.

What happens when you click Push:
1. Reads your current product data from this panel
2. Generates updated script.js content
3. Commits it to GitHub via API
4. Vercel detects the commit and auto-deploys (~30 sec)
5. Your live website is updated ✅

Export & Deploy

Since your website is static HTML, changes made here are saved in this browser. Use the steps below to apply changes to your live site at qualitypharmaceuticals.co.in.

1

Click Download script.js below — this generates a new script file with all your product edits.

2

Log in to your hosting control panel (cPanel / FTP / File Manager).

3

Navigate to your website root and replace the existing script.js with the downloaded file.

4

Hard-refresh your website (Ctrl+Shift+R) to see the changes live.
